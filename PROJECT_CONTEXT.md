Project: HAEFA Progoty



Purpose:

Rural healthcare clinical workflow platform designed for low-resource clinics.



Technology Stack:

React

TypeScript

Vite

Firebase Authentication

Cloud Firestore

Firebase Storage

Firebase Cloud Functions

Material UI



Primary Workflow:

Registration → Vitals → Doctor → Pharmacy



Core Entities:

Patients

Encounters

Queue

Medications

Prescriptions



User Roles:

registration

nurse

doctor

pharmacy

admin



Primary Goal:

Enable small rural clinics to run structured patient workflows using inexpensive tablets or laptops.



Design Principles:

Simple UI

Fast patient movement

Minimal typing

Support for clinics with no internet (future)

Safe medication prescribing
